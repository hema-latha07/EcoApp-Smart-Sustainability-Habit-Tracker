🌿 EcoApp - Smart Sustainability & Habit Tracker
EcoApp is an interactive, gamified Android application designed to help users build eco-friendly habits and live a more sustainable lifestyle. Built natively for Android using Java and XML, the app challenges users to log daily green tasks, tracks their long-term environmental impact, and provides curated knowledge to foster eco-conscious decisions.

🚀 Features
Goal-Oriented Onboarding: Customizes the user experience based on chosen green archetypes (e.g., The Commuter, The Foodie, The Minimalist).

Smart Camera Verification: Combats "cheating" by prompting users to take context-aware photo evidence of their green task (e.g., cycling gear, plastic-free meals) before awarding points.

Simulated AI Image Analysis: Features an interactive "AI Analysis" loading phase to mimic cloud-based proof verification.

Gamified Progression System: Converts tasks into Eco-Points that fill up a progress bar, evolving the user's rank from a Seedling 🌱 to a Sapling 🌿 and ultimately an Eco-Warrior 🌳.

Persistent Progress: Uses local data storage (SharedPreferences) to ensure user profiles, ranks, and points are securely saved across app sessions.

Interactive Navigation Grid: An elegant dashboard layout allowing seamless exploration into core feature hubs: Green Academy, Eco Quiz, Eco-Map, and personal Impact Analytics.

🛠️ Tech Stack & Architecture
Frontend: XML layout files leveraging advanced View Groups (GridLayout, ScrollView, CardView) for a clean, modern UI/UX.

Backend Logic: Java utilizing modern API patterns like Android Jetpack's ActivityResultLauncher and asynchronous handlers for simulated real-time event delays.

Data Persistence: Key-value data architecture with Android SharedPreferences.
